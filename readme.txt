Connect your WordPress site to AI SEO Tool for AI citation readiness audits directly within your admin dashboard.

== Description ==

AI SEO Tool lets WordPress site owners run a full GEO (Generative Engine Optimization) citation readiness audit of their site, directly from the WordPress admin.

Features:

* Connect to your AI SEO Tool account via API key
* Trigger full site audits with one click
* View GEO Citation Score, per-engine scores, and improvement suggestions
* Review technical audit results (HTTPS, sitemap, broken links, security headers, PageSpeed)
* Track audit history over time

== Installation ==

1. Upload the `ai-seo-tool` folder to `/wp-content/plugins/`
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to AI SEO Tool > Settings and enter your API key
4. Click "Connect Account" to link your AI SEO Tool account

== Frequently Asked Questions ==

= Where do I get an API key? =

Log in to your AI SEO Tool account, go to Settings > API Keys, and generate a new key.

= What plans are supported? =

Free plan users can view the Dashboard tab. Pro and Agency plan users have access to all features including GEO Analysis, Technical Audit, and History.
